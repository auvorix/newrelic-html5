About

This page exists to verify that New Relic correctly tracks navigation between pages on a static HTML5 site hosted on Azure Static Web Apps.

About this project

A minimal static HTML5 site with three pages. Deployed automatically on every push to the main branch via GitHub Actions → Azure Static Web Apps.

Technology stack

HTML5 semantic markup + CSS — no JavaScript framework, no build step. New Relic is injected via the copy/paste script snippet placed in the <head> of every page.

Why semantic HTML5?

Semantic tags like <main>, <article>, and <nav> describe the purpose of content to browsers, search engines, and screen readers — making the page more accessible and SEO-friendly.
