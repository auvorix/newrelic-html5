Console Lab

Trigger real HTTP status codes and JavaScript errors — observe them in the browser console and in the log below.

How to use:
Open DevTools → Console (F12), then click any button. Each request or error appears both here and in the console.

Status ranges

2xx — Success
3xx — Redirection
4xx — Client Error
5xx — Server Error
JS — Runtime Error
Tip:
In the Console tab, use the filter buttons to show only Errors or Warnings. The Network tab shows every fetch with full response headers.

2xx — Success

Request received, understood and accepted.

3xx — Redirection

Further action is required to complete the request.

4xx — Client Error

The request contains bad syntax or cannot be fulfilled.

5xx — Server Error

The server failed to fulfil an apparently valid request.

JavaScript Errors

Runtime errors thrown directly in JavaScript — no network request involved.
