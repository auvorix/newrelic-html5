Home Page

This is a test site for New Relic Browser Monitoring using HTML5 semantic elements — <header>, <nav>, <main>, <article>, and <footer>.

What this site tests

Page load time, JavaScript errors, AJAX requests, and user sessions — all tracked by the New Relic browser agent injected via the copy/paste snippet.

HTML5 vs basic HTML

The difference is structural. HTML5 uses semantic tags that describe the meaning of content (header, nav, main, article, footer). Both approaches work identically with New Relic.

Navigate the pages

Each page visit is tracked as a separate page view event in New Relic. Navigate to About and Contact to generate multiple data points.
